# Bài nghiên cứu/luận văn nháp bằng LaTeX

File chính:

```text
research/main.tex
```

Nội dung bám theo đề tài:

```text
Nghiên cứu giải pháp phát hiện và phòng chống tấn công API trong hệ thống Microservices dựa trên OWASP API Security Top 10
```

Định dạng theo hướng dẫn trình bày đề cương/luận văn:

- A4.
- Times New Roman.
- Cỡ chữ 14pt.
- Giãn dòng 1.5.
- Lề trên 3cm, dưới 2cm, trái 3cm, phải 2cm.
- Thụt đầu dòng 1cm.

Biên dịch:

```bash
xelatex main.tex
xelatex main.tex
```

Máy hiện tại chưa có `xelatex/pdflatex/lualatex`, nên chưa xuất PDF trực tiếp tại đây.
